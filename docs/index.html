すきなプレイスタイルで。

鼻毛鯖のルールは一つだけ。それは、チート（ずる）をしないこと。 建築を壊したり、アイテムを取ったりしてもルール違反になることはありません。すきなことをして遊ぼう！

自衛はしっかりと！

自由なサーバーだから、悪意をもったプレイヤーがいるかも・・・？保護機能を使って、建築はしっかりと保護しよう。保護するために必要なアイテムは最初から持っているよ。

さあ、今すぐはじめよう。
